# Contributing

Thank you for your interest in contributing to Cloudbeds ID Scanner!

## How to contribute

### Reporting bugs

If you find a bug, please open an issue including:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if possible
- Chrome version

### Suggesting improvements

Suggestions are welcome! Open an issue with:
- Description of the improvement
- Use case
- Possible implementation (if you have one)

### Pull Requests

1. Fork the repository
2. Create a descriptive branch: `feature/my-improvement` or `fix/bug-description`
3. Make atomic commits with clear messages
4. Make sure the code works correctly
5. Open a PR with a detailed description

## Code style

- Use tabs for indentation
- Variable and function names in camelCase
- Keep consistency with existing code

## Development environment setup

1. Clone the repository
2. Load the extension in Chrome (developer mode)
3. Make changes
4. Reload the extension to test

## Questions

If you have questions, open an issue with the `question` label.
